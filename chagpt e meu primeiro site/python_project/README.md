# Meu Treino - CLI em Python

Pequeno utilitário CLI para gerenciar hábitos localmente.

Como usar

1. Garantir Python 3.8+
2. Tornar executável (opcional): `chmod +x app.py`
3. Executar:

```powershell
python app.py list
python app.py add "Ler 30 minutos"
python app.py done 1
python app.py clear
```

Os dados são salvos em `python_project/habits.json`.
